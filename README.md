## Installation


#### 1. In the section repositories of your composer.json, add:

```
{
    "type": "vcs",
    "url": "https://github.com/thememachine/eleven"
}
```

#### 2. Install theme

```
composer require thememachine/eleven
```

Afterwards enable modules as usual.
